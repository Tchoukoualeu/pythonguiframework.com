Kivy

Modern multi-touch framework

Overview

Kivy is a modern, multi-touch-enabled Python framework for building innovative user interfaces. It supports various input devices and is particularly well-suited for interactive applications. Key Features: • Multi-touch support • GPU acceleration • Cross-platform • Rich widget library • Custom shader support Common Use Cases: • Mobile applications • Games • Interactive installations • Multi-touch applications • Visualization tools

Installation

pip install kivy

Example

from kivy.app import App
from kivy.uix.label import Label

class HelloApp(App):
    def build(self):
        return Label(text='Hello, World!')

HelloApp().run()
